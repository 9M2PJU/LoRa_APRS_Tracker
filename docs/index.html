9M2PJU Mod LoRa APRS Tracker

Firmware Web Installer — Heltec Wireless Tracker

Heltec Wireless Tracker (ESP32-S3 + ST7735 TFT)

Click the button above, select your device's serial port,
and flash the firmware directly from your browser.

Browser not supported

This installer requires a Chromium-based browser with Web Serial support (Google Chrome, Microsoft Edge, or Opera). Firefox and Safari do not support Web Serial API. Please switch to Chrome or Edge.

What this firmware includes

  • Callsign: 9M2PJU-7 (all 3 beacon profiles)
  • LoRa frequency: 433.400 MHz (Malaysia)
  • LoRa region label: LoRa[MY]
  • Display clock: Malaysia Time (UTC+8)
  • Military green header bar on all screens
  • Color-coded body text (GPS, battery, altitude, messages)
  • Colored APRS symbol by category
  • Status accent bar (GPS lock / BT / idle)
  • Startup branding: "9M2PJU Mod"
  • GPS Eco Mode enabled (battery saving)
  • Profile 3 symbol: motorcycle

How to install

  • Use Google Chrome or Microsoft Edge (not Firefox/Safari)
  • Connect your Heltec Wireless Tracker via USB cable
  • Click the "CONNECT" button above
  • Select the serial port (usually "USB JTAG/serial debug unit")
  • Click "Install" and wait for the flash to complete
  • Your tracker will reboot automatically with the new firmware

After flashing

  • The config file is pre-configured for Malaysia (9M2PJU-7, 433.400 MHz)
  • To change settings: enable WiFi AP mode and visit 192.168.4.1
  • To reflash SPIFFS config separately, use PlatformIO: pio run -e heltec_wireless_tracker -t uploadfs
